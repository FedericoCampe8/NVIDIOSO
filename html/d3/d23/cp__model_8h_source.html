8 // This class represents a genral Constraint Programming model.
9 // It holds the state concerning all the information needed to
10 // explore the search space in order to find solutions.
11 // It holds the following finite sets:
12 // a) V of Variables, each associated with a finite domain D
13 // b) C of constraints over the variables in V.
14 // This set is represented by the correspondent Constraint Store.
15 // This class represents general objects.
16 // Specific implementations of the solver (i.e., using the CUDA framework)
17 // should derive from this class and specilize the following methods.
18 //
19 // @note: this class is also used to generate the constraint graph
