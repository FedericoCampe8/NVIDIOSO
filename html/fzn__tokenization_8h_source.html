8 // This tokenizer is specialized for FlatZinc file
9 // It specializes tokenization.
10 // The tokenizer splits a string in tokens and manages them
11 // in order to create the appropriate tokens representing
12 // objects in the model.
13 // For example, given the string:
14 // "array [1..2] of var set of 1..3: a;"
15 // the tokenizer splits it into substrings and
16 // performs different actions according to the substring:
17 // - "array" : instantiates a token_arr;
18 // - "[1..2]" : sets the size of the array;
19 // - "set" : defines the type of domain;
20 // - "1..3" : defines the type of set-domain;
21 // - "a" : defines the name (id) of the array.
22 //
