8 // This class specializes the Constraint class with the constraints
9 // as defined by the Specification of the FlatZinc language.
10 // The interested reader can consult
11 // http://www.minizinc.org/downloads/doc-1.3/flatzinc-spec.pdf
12 // for the complete list of constraints and their specification.
13 //
14 // @note To implement a constraint at least the following methods
15 // should be specialized:
16 // - constructor
17 // - setup method
18 // - consistency
19 // - satisfied
20 // - print_semantic
21 // @note Remember to set the size of the scope in the constructor
22 // according to the number of parameters given in input.
23 // @note Remember to perform consistency checking on the pointers
24 // given in input to each propagator.
25 // @note See int_ne.h and int_ne.cpp for an example of how to write
26 // a propagator.
27 // @note If at least one FD variable involved in the constraint has
28 // an empty domain and the rest of the variables satisfy the
29 // the constraint, than the constraint is trivially satisfied.
30 //
