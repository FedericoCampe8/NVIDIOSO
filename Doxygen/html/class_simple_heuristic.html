SimpleHeuristic::SimpleHeuristic ( std::vector< Variable * >  vars,
VariableChoiceMetric *  var_cm,

Constructor, defines a new simple heuristic given the metrics for selecting the next variable to label and the value to assign to such variable.

Parameters
varsa vector of pointer to variables to label.
var_cmthe variable metric used to select the next variable to label.
val_cmthe value metric used to select the next value to assign to the selected variable.
Note
if the variable metric is a nullptr, the next variable to label is the first non-ground variable.
